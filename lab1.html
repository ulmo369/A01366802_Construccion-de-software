  3. ¿Cuál es el propósito de los métodos HTTP: GET, HEAD, POST, PUT, PATCH, DELETE?
    • GET: Solicita una representación del recurso especificado. Debe usarse para recuperar datos
    • HEAD: Provee información general (metadatos) acerca del documento (Titulo, script, etc)
    • POST: Envía una entidad a un recurso
    • PUT: Crea un nuevo elemento o reemplaza una representación del elemento de destino con los datos de la petición
    • PATCH: Aplica modificaciones a un recurso
    • DELETE: Borra un recurso
    Fuente: www.freecodecamp.org

  10. ¿Qué componentes de estructura y estilo tiene una tabla?
    • table: Son datos con más de una dimensión.
    • caption: Dar un título a tu tabla
    • col: Columna de una tabla
    • colgroup: Grupo de columnas
    • thead: Delimitar el encabezado de la tabla
    • tfoot: Delimita la parte de la tabla correspondiente al pie de página
    • tbody: delimita las otras partes del contenido de la tabla que no están en el encabezado o en el pie de página de la tabla
    • td: Celda con datos de una tabla
    • tr: Fila de celdas en una tabla
    • th: Celda encabazado de una tabla
    Fuente: www.htmlquick.com

  1. ¿Qué diferencias y semejanzas hay entre Java y JavaScript?
    • JavaScript se ejecuta esencialmente a través de tu navegador web. Por otro lado, Java es un lenguaje compilado: pasa por una JVM (Java Virtual Machine) para ser interpretado y traducido al lenguaje de la «máquina».


  2. ¿Qué métodos tiene el objeto Date? (Menciona al menos 5*)
    • Date.now() = Devuelve el valor numérico correspondiente a la hora actual.
    • Date.parse() = Transforma una cadena que representa una fecha, y devuelve el número de milisegundos desde el 1 de enero de 1970, hora local 00:00:00.
    • Date.UTC() = Acepta los mismos parámetros que la forma más larga del constructor, y devuelve el número de milisegundos en un objeto Date desde el 1 de enero de 1970, hora universal 00:00:00.
    • getTime() = Regresa el tiempo expecifico de un momento.
    • getMonth() = Regresa el mes.
    Fuente: developer.mozilla.org

  3. ¿Qué métodos tienen los arreglos? (Menciona al menos 5*)

    • join() = Junta los elementos de un array en una cadena con un separador.
    • lastIndexOf()= Busca un elemento en un array y devuelve su posición. Busca desde el final.
    • pop() = Borra el último elemento del array
    Fuente: tutobasico.com
